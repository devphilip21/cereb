title: Gesture

Gesture is the highest layer in Cereb, responsible for recognizing meaningful user intentions from signal streams. It transforms low-level input data into semantic gesture events.

## What is a Gesture?

A gesture represents a user's intent:
- **Pan**: Dragging movement
- **Pinch**: Two-finger zoom (planned)
- **Rotate**: Two-finger rotation (planned)
- **Swipe**: Quick directional movement (planned)

## Gesture Events

Gesture events extend the signal phase model with gesture-specific data:

```typescript
interface GestureEvent {
  phase: "start" | "change" | "end" | "cancel";
  timestamp: number;
  target: EventTarget;
}

// Pan extends this with movement data
interface PanEvent extends GestureEvent {
  x: number;
  y: number;
  deltaX: number;
  deltaY: number;
  velocityX: number;
  velocityY: number;
}
```

## Using Pan Gesture

```typescript
import { singlePointer } from "@cereb/single-pointer";
import { pan } from "@cereb/pan";
import { pipe } from "@cereb/core";

const element = document.getElementById("draggable")!;
const pointer$ = singlePointer(element);
const pan$ = pipe(pointer$, pan());

pan$.subscribe((event) => {
  if (event.phase === "change") {
    element.style.transform = `translate(${event.deltaX}px, ${event.deltaY}px)`;
  }
});
```

## Preventing Default Behavior

Use `PREVENTED` to stop gesture propagation:

```typescript
import { PREVENTED, isPrevented } from "@cereb/core";

pan$.subscribe((event) => {
  if (someCondition) {
    return PREVENTED; // Stop further processing
  }
});
```
