---
title: tap
description: Perform side effects without modifying values
---

Performs a side effect for each emission without modifying the values.

## Signature

```typescript
function tap<T>(callback: (value: T) => void): Operator<T, T>
```

## Parameters

| Name | Type | Description |
|------|------|-------------|
| `callback` | `(value: T) => void` | Function to call for each value |

## Usage

```typescript
import { pipe, from } from "@cereb/core";
import { tap, map } from "@cereb/core";

const result$ = pipe(
  from([1, 2, 3]),
  tap((x) => console.log("before:", x)),
  map((x) => x * 2),
  tap((x) => console.log("after:", x))
);

result$.subscribe();
// Output:
// before: 1
// after: 2
// before: 2
// after: 4
// before: 3
// after: 6
```

## Real-World Example

```typescript
import { pan } from "@cereb/pan";
import { pipe } from "@cereb/core";
import { tap } from "@cereb/core";

const pan$ = pipe(
  singlePointer(element),
  pan(),
  tap((event) => {
    // Log for debugging without affecting the stream
    console.log("Pan event:", event.phase, event.deltaX, event.deltaY);
  })
);
```

## Use Cases

- Debugging: Log values without modifying them
- Analytics: Track events
- DOM updates: Update UI as a side effect
