---
title: debounce
description: Delay emission until quiet period
---

Emits a value only after a specified time has passed without another emission.

## Signature

```typescript
function debounce<T>(ms: number): Operator<T, T>
```

## Parameters

| Name | Type | Description |
|------|------|-------------|
| `ms` | `number` | Debounce duration in milliseconds |

## Usage

```typescript
import { pipe, fromEvent } from "@cereb/core";
import { debounce, map } from "@cereb/core";

const input = document.getElementById("search")!;
const input$ = fromEvent(input, "input");

// Wait 300ms after user stops typing
const search$ = pipe(
  input$,
  debounce(300),
  map((e) => (e.target as HTMLInputElement).value)
);

search$.subscribe((query) => {
  performSearch(query);
});
```

## throttle vs debounce

| Operator | Behavior |
|----------|----------|
| `throttle` | Emits first, ignores rest for duration |
| `debounce` | Waits until quiet, then emits last |

```
Events:  --a-b-c------d--e--------
throttle(50): a--------d-----------
debounce(50): ------c--------e-----
```

## Real-World Example: Resize Handler

```typescript
import { fromEvent, pipe } from "@cereb/core";
import { debounce } from "@cereb/core";

const resize$ = pipe(
  fromEvent(window, "resize"),
  debounce(150)
);

resize$.subscribe(() => {
  // Only called once resizing stops
  recalculateLayout();
});
```
