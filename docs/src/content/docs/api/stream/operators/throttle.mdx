---
title: throttle
description: Rate-limit emissions
---

Emits a value, then ignores subsequent values for a specified duration.

## Signature

```typescript
function throttle<T>(ms: number): Operator<T, T>
function throttleLast<T>(ms: number): Operator<T, T>
```

## Parameters

| Name | Type | Description |
|------|------|-------------|
| `ms` | `number` | Throttle duration in milliseconds |

## throttle vs throttleLast

- `throttle`: Emits the **first** value in each window
- `throttleLast`: Emits the **last** value in each window

## Usage

```typescript
import { pipe, fromEvent } from "@cereb/stream";
import { throttle } from "@cereb/stream/operators";

const scroll$ = fromEvent(window, "scroll");

// Emit at most once every 100ms
const throttled$ = pipe(
  scroll$,
  throttle(100)
);
```

## Real-World Example: 60fps Animation

```typescript
import { singlePointer } from "@cereb/single-pointer";
import { pipe } from "@cereb/stream";
import { throttle } from "@cereb/stream/operators";

const pointer$ = singlePointer(element);

// Limit to ~60fps (16.67ms)
const smooth$ = pipe(
  pointer$,
  throttle(16)
);

smooth$.subscribe((signal) => {
  updatePosition(signal.x, signal.y);
});
```

## throttleLast Example

```typescript
import { throttleLast } from "@cereb/stream/operators";

// Get the last position in each 100ms window
const lastPosition$ = pipe(
  pointer$,
  throttleLast(100)
);
```
