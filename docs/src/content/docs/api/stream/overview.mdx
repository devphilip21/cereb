`@cereb/stream` is a lightweight, tree-shakeable Observable implementation.

## Installation

```bash
pnpm add @cereb/stream
```

## Core Types

### `Observable<T>`

A lazy, push-based data source.

```typescript
interface Observable<T> {
  subscribe(observer: Observer<T> | ((value: T) => void)): Unsubscribe;
}
```

### `Observer<T>`

Consumes values from an Observable.

```typescript
interface Observer<T> {
  next: (value: T) => void;
  error?: (error: unknown) => void;
  complete?: () => void;
}
```

### `Operator<T, R>`

Transforms an Observable.

```typescript
type Operator<T, R> = (source: Observable<T>) => Observable<R>;
```

## Factory Functions

### `createObservable<T>(subscribe)`

Creates a custom Observable.

```typescript
const timer$ = createObservable<number>((observer) => {
  let count = 0;
  const id = setInterval(() => observer.next(count++), 1000);
  return () => clearInterval(id);
});
```

### `fromEvent(target, type)`

Creates an Observable from DOM events.

```typescript
const click$ = fromEvent(button, "click");
```

### `from(iterable)`

Creates an Observable from an array or iterable.

```typescript
const numbers$ = from([1, 2, 3]);
```

### `of(...values)`

Creates an Observable that emits specified values.

```typescript
const values$ = of(1, 2, 3);
```

### `interval(ms)`

Emits sequential numbers at specified interval.

```typescript
const tick$ = interval(1000);
```

## Subject

A Subject is both Observable and Observer.

### `createSubject<T>()`

```typescript
const subject = createSubject<number>();

subject.subscribe((v) => console.log(v));
subject.next(1); // logs: 1
```

### `createBehaviorSubject<T>(initial)`

Subject with current value.

```typescript
const state = createBehaviorSubject(0);
console.log(state.value); // 0

state.subscribe((v) => console.log(v)); // immediately logs: 0
state.next(1); // logs: 1
```

## Pipe

Use `pipe` to compose operators:

```typescript
import { pipe } from "@cereb/stream";
import { filter, map } from "@cereb/stream/operators";

const result$ = pipe(
  source$,
  filter((x) => x > 0),
  map((x) => x * 2)
);
```
